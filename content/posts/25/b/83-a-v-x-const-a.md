+ Acceleration
$$\tag{C1}
a(t) = a_n, \ \ \ \ t_n < t < t_{n+1}.
$$
+ Velocity
$$\tag{C2}
v(t) = v_n + a_n(t - t_n), \ \ \ \ t_n \le t \le t_{n+1}.
$$
+ Posisition
$$\tag{C3}
x(t) = x_n + v_n(t - t_n) + \tfrac12 a_n(t - t_n)^2, \ \ \ \ t_n \le t \le t_{n+1}.
$$
